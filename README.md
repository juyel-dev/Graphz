# Graphz - Interactive Science Graph Explorer

![Graphz Logo](assets/icons/icon-192x192.png)

A modern, mobile-first web application for exploring interactive graphs and diagrams in Physics, Chemistry, and Mathematics.

## 🚀 Features

### Core Features
- **Hybrid Graph System**: Interactive JavaScript graphs + Static image diagrams
- **Smart Search**: AI-like search with typo correction and suggestions
- **Mobile-First**: Optimized for mobile devices with touch-friendly interface
- **Admin Panel**: Complete content management system
- **Ad System**: Managed advertisement zones
- **PWA Ready**: Installable as mobile app with offline functionality

### Graph Types
- **Physics**: Kinematics, Electricity, Waves, Thermodynamics
- **Chemistry**: Kinetics, Equilibrium, Electrochemistry, Organic
- **Mathematics**: Algebra, Calculus, Trigonometry, Geometry

### Interactive Features
- Real-time parameter controls
- Multiple graph comparisons
- Custom equation input (future)
- Zoom and pan capabilities

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: LocalStorage (with export/import)
- **Graphics**: Canvas API, SVG
- **PWA**: Service Workers, Web App Manifest
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Inter)
